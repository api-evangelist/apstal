# Apstal

AI-first website analytics and visitor tracking platform that replaces traditional dashboards with a natural-language chatbot interface, offering conversational analytics, cookieless tracking, session replays, and heatmaps. Programmatic access is provided through a hosted MCP server exposing analytics tools.
